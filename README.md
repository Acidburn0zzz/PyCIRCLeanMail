# PyCIRCLeanMail

PyCIRCLeanMail is the Python code mased on [CIRCLean](https://www.circl.lu/projects/CIRCLean/), but to sanitize emails.

# Installation

~~~
python setup.py install
pip install -r requirements.txt
~~~
